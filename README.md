# Project Title
Three Blind Mice
## Project Description
This is a main menu for a late 90's point-and-click adventure game.

## Intended User or Player
Designed for all ages, particualrly to be easily understandable for young children.

## User / Player Goal
Children should be able to easily see and click each button, and not gte bogged down by unintelligble or hard to understand settings; just make it easy to get right into the game.

## Inspiration Interfaces
The interfaces I drew from, including the Jumpstart Artist and Reader Rabbit PC game menus, showed me how I wanted to visually make my buttons stands out with bright colors that highlighted themselves when moused over, as well as keeping everything simple and clean.

## Design Choices
- Layout: Menu buttons are aligned horizontally next to each other
- Color: Mainly bright and pale yellows, black text
- Typography: Plain Arial for menus, specialized drawn font for logo
- Visual hierarchy: Logo, menu buttons, followed by key art and copyright text, in that order top to bottom
- Feedback states: Buttons highlight on mouse-over
- Responsive design: Buttons slign vertically on smaller screens
